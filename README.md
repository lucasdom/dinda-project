# DindaProject

## Rodar o projeto

Para rodar o projeto, execute o seguinte comando na pasta raiz do projeto:

```ruby
ruby main.rb
```

## Testar o projeto
Para testar o projeto, execute o seguinte comando:

```ruby
rspec spec
```
